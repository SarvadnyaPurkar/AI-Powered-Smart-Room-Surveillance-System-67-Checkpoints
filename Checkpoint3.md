# Checkpoint 3: Voice Command System Responds to Basic Commands
## Goal

Control the system using voice.

## Commands:

1) Start system
2) Stop system
3) Take snapshot
## Topics to Learn
### Speech Recognition Basics

### Understand:

1) Audio input
2) Microphone handling
3) Speech-to-text
### Command Parsing

1) Map recognized text to actions.

### Mini Tasks
1) Detect microphone input
2) Print recognized speech
3) Trigger commands
4) Add start/stop logic (eg print "System is started" if voice commands start and print "System stopped" when voice commands stop)
## Expected Output

System starts/stops using voice commands.
